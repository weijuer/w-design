# W.Design Vue

一个基于 Vue 3 的组件库，提供简洁、优雅的 UI 组件。

## 快速开始

### 安装

```bash
npm install w-design-vue
# 或
yarn add w-design-vue
# 或
pnpm add w-design-vue
```

### 引入

```js
import { createApp } from 'vue'
import App from './App.vue'
import WDesignVue from 'w-design-vue'


const app = createApp(App)
app.use(WDesignVue)
app.mount('#app')
```

### 按需引入

```js
import { WButton } from 'w-design-vue'

```
